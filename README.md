- python-nmap

```
pip install python-nmap
```

Download - [python-nmap-0.6.1.tar.gz](https://xael.org/pages/python-nmap-0.6.1.tar.gz)


```
tar xvzf python-nmap-0.6.1.tar.gz
cd python-nmap-0.6.1
python setup.py install
```
